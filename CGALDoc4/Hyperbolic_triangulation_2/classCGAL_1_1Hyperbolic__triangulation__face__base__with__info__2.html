Definition

The class Hyperbolic_triangulation_face_base_2 is designed as one of the default models for the concept HyperbolicTriangulationFaceBase_2.

Template Parameters
Infois the information the user wants to add to a face. Must be DefaultConstructible and Assignable.
Gtmust be a model of HyperbolicDelaunayTriangulationTraits_2.
Fbmust be a model of HyperbolicTriangulationFaceBase_2. Defaults to Hyperbolic_triangulation_face_base_2<GT>.
