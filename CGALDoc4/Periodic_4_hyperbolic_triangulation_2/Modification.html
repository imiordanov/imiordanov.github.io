The Bolza surface is a closed compact orientable surface of genus 2. A closed compact surface of genus at least 2 is conformally equivalent to some surface of constant negative curvature, therefore it is hyperbolic. Among all surfaces of genus 2, the Bolza surface is the most symmetric one, and its properties are well understood. It is a natural extension from genus 1 (the Euclidean flat torus, see the documentation for 2D Periodic Triangulations Reference) to genus 2.

A triangulation is a simplicial complex, i.e.:

See Figure 42.1 - Left.

The four translations \(a, b, c, d\) generate a discrete group of orientation-preserving isometries, with finite presentation

\[ \mathcal{G} = \left< a,b,c,d \; \bigg| \; abcd\overline{a}\overline{b}\overline{c}\overline{d} \right>. \]

The Bolza surface \(\mathcal{M}\) is defined as \(\mathcal{M} = \mathbb H^2 / \mathcal{G}\). \(\mathcal G\) is the fundamental group of \(\mathcal M\). The octagon \(\mathcal D_O\) is a fundamental domain for \(\mathcal G\).

The half-open octagon shown in Figure 42.1 - Right contains exactly one representative of each point on \(\mathcal{M}\), and we call that half-open octagon the original domain \(\mathcal D\). See Figure 42.2 for an illustration of the construction of a genus-2 surface from \(\mathcal D_O\).

From now on, whenever we talk about a point, we will generally mean the representative of an element of \(\mathcal{M}\) that lies inside \(\mathcal D\).

Note, however, that the removal of a vertex is authorized only if the triangulation remains valid (as defined above). If this condition is not satisfied, then the vertex is not removed from the triangulation.

For instance, a single point does not define a triangulation of \(\mathcal M\), as the result would not be a simplicial complex. For this reason, we initialize a triangulation of \(\mathcal M\) with a predetermined set of 14 points, called dummy points, whose triangulation in \(\mathcal M\) is a simplicial complex and has the empty circle property. The set of dummy points has been proposed in [1].

See Figure 42.4

After the triangulation has been initialized with the dummy points, new input points can be inserted without ever losing the simplicial complex property. If sufficiently many well-distributed points are inserted, the dummy points become unnecessary, i.e., the triangulation remains a simplicial complex even if we remove them. If the set of input points is not sufficiently large, or if the points are not well-distributed on the surface, then the final triangulation will contain the dummy points necessary to guarantee that the triangulation is a simplicial complex. Experiments have shown that, on average, all dummy points are removed from the triangulation with the insertion of less than 200 random points uniformly distributed (according to the Euclidean metric) in the unit disk.

The two main classes Periodic_4_hyperbolic_Delaunay_triangulation_2 and Periodic_4_hyperbolic_triangulation_2 provide high-level geometric functionality and are responsible for the geometric validity of the triangulation. Periodic_4_hyperbolic_Delaunay_triangulation_2 contains all the functionality that is specific to Delaunay triangulations, such as point insertion and vertex removal, the side-of-circle test, finding the conflicting region of a given point, and dual functions. Periodic_4_hyperbolic_triangulation_2 contains all the functionality that is common to triangulations in general, such as location of a point in the triangulation [3], and access functions. Note that Periodic_4_hyperbolic_triangulation_2 does not support point insertion or vertex removal.

This parameter must meet the requirements described in the concept Periodic_4HyperbolicDelaunayTriangulationTraits_2. For more details, see Section The Geometric Traits Parameter.
